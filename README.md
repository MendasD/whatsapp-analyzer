# whatsapp-analyzer

A Python package for automatic analysis and topic classification of WhatsApp
group conversations, developed as part of a Web Scraping course project (2025–2026).
It processes native WhatsApp exports entirely locally — no data is sent to any server.

---

## Installation

```bash
# Clone and install in editable mode with dev dependencies
git clone https://github.com/<your-org>/whatsapp-classifier.git
cd whatsapp-classifier
pip install -e ".[dev]"

# Download spaCy language models (required for French/English lemmatisation)
python -m spacy download fr_core_news_sm
python -m spacy download en_core_web_sm

# Optional: NLTK stopwords
python -m nltk.downloader stopwords

# Optional: media analysis (Whisper audio transcription, image processing)
pip install -e ".[media]"
```

## Quick start

### Python API

```python
from whatsapp_analyzer import WhatsAppAnalyzer, GroupComparator

# Analyse a single group
az = WhatsAppAnalyzer("chat.zip")          # also accepts .txt or a folder
az.parse().clean().analyze(topics=5, sentiment=True)

az.topics()                                # dominant topics
az.users()                                 # list of participants
az.user("Aminata").summary()              # full profile for one user
az.user("Aminata").sentiment_over_time()  # sentiment evolution
az.timeline()                             # activity over time
az.report(output="rapport/")             # export HTML report

# Compare multiple groups
comp = GroupComparator(["group1.zip", "group2.zip", "group3.zip"])
comp.compare_topics()
comp.compare_activity()
comp.common_users()
comp.report(output="comparaison/")
```

### CLI

```bash
# Analyse a group
whatsapp-analyzer analyze --input chat.zip --topics 5 --sentiment

# Include media analysis (requires [media] extras)
whatsapp-analyzer analyze --input chat.zip --media

# Focus on one participant
whatsapp-analyzer analyze --input chat.zip --user "Aminata"

# Compare multiple groups
whatsapp-analyzer compare --inputs g1.zip g2.zip g3.zip

# Launch the web interface
whatsapp-analyzer serve
```

### Streamlit web interface

```bash
whatsapp-analyzer serve
# Opens at http://localhost:8501
```

Upload a `.zip` or `.txt` export, configure the analysis parameters,
and download the generated HTML report — no coding required.

| Folder | Already-decompressed export directory |

## Legal and privacy

- Exports are generated by the user from their own groups — no illegal scraping.
- All processing is **local**: no message content is sent to any external server.
- RGPD compliance: anonymisation of names and phone numbers is available via
  the `--anonymize` flag.
- Strictly personal or academic use — redistribution of raw chat data is prohibited.
- Complies with WhatsApp (Meta) Terms of Service: no automation of the WhatsApp
  interface is involved.
